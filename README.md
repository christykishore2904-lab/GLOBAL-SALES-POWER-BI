# 🌍 Global Sales Dashboard - Power BI

A comprehensive **Power BI dashboard** that analyzes global sales performance across regions, products, and time periods. This project helps businesses gain insights into sales trends, profitability, and regional performance.

---

## 📊 Project Overview

The **Global Sales Dashboard** provides interactive visualizations to track key sales metrics and business performance worldwide.

The dashboard allows users to:
- Monitor total sales and profit
- Analyze regional performance
- Identify top-selling products
- Track sales trends over time
- Evaluate category and segment performance

---

## 🛠 Tools & Technologies

- **Power BI Desktop**
- **Microsoft Excel / CSV Dataset**
- **Data Modeling**
- **DAX (Data Analysis Expressions)**
